出品 | 搜狐科技作者 | 林国振近日，OPPO正式发布新款OPPO Find X2系列手机，全系搭载了骁龙865处理器、120Hz 2K屏幕。其中OPPO Find X2 8GB+128GB售价5499元，Find X2 8GB+256GB售价5999元，Find X2 Pro 12GB+256GB售价6999元，Find X2夜海/碧波和Find X2 Pro缎黑/茶橘已正式开售。搜狐科技已经拿到了OPPO Find X2 Pro茶橘版本，下面就为大家带来上手体验。OPPO Find X2系列依旧延续了此前Find系列的高颜值和高性能的设计思路。在外观设计上，Find X2 Pro采用了正反双曲面的设计。正面为康宁大猩猩第六代玻璃面板，利用先进的COP封装工艺，将下边框及左右边框分别收窄至 2.98mm 和 1.4mm，配合 67.8°曲面屏设计，视觉上呈现无边界的效果。OPPO Find X2背面除了玻璃材质之外，还提供陶瓷与皮革两种材质，不仅提升了产品颜值的同时，也带来了更好的质感。Find X2拥有碧波、夜海、茶橘三款配色，Find X2 Pro则拥有缎黑、茶橘、雅灰三款配色。对于普通用户来说，屏幕参数的升级对感知的提升可能并不明显。但是OPPO Find X2系列的屏幕刷新率提升对于视觉感受的改变却是实实在在的。Find X2全系配备OPPO定制的120Hz、QHD+、10亿色超感屏，从分辨率、色彩、刷新率、亮度等方面带来全方位视觉革命。其分辨率达3168*1440，513ppi，局部峰值亮度1200尼特，对比度达5000000:1，超低屏幕反射率，HDR10+认证，暗光、强光下显示更加清晰通透。无论是浏览网页、欣赏图片还是观看高清HDR视频体验都有巨大飞跃，对游戏爱好者来说也是巨大的福利。更高的刷新率意味着在同样的时间内，有着更高的帧数变化，画面残影更少，整体的动画运动过程更为细腻丰满。手机上常用“Hz”表示这块屏幕的刷新率，传统的60Hz屏幕意味着一秒钟刷新60次，而OPPO Find X2 Pro屏幕的刷新率为120Hz，意味着一秒钟刷新120次。那120Hz的刷新率在使用上又会有怎样的改变呢？首先我们得知道，人眼对于帧率的识别是没有上限的。而相较于60Hz来说，120Hz的屏幕在使用上会更为流畅。所以，用户在使用120Hz高帧率屏幕后，再去看60Hz刷新率的屏幕，就会变得很难适应。此外，OPPO还发布旗下首款O1超感画质引擎，内置独立显示芯片，为120Hz超感屏打造高品质内容，增强视频画质表现。Find X2系列首发支持Motion Clear视频动态插帧和HDR视频画质增强技术。前者可将30帧以下在线影视、体育直播等视频内容提升至60甚至120帧，降低视频拖影和运动抖动，增强清晰度和流畅度。后者可对主流视频、短视频平台内容进行实时HDR效果增强，通过色调映射、位深扩展、色域扩展等多种图像增强算法，提升普通视频的画面质感。此外，OPPO与第三方知名游戏公司合作，推出120fps高帧率游戏大作，充分发挥Find X2系列120Hz超感屏的实力。可以看出，此次Find X2在屏幕体验的提升是务实的，不止在于硬件，更通过软硬结合的方式，让观看体验得到质的提升。Find X2 pro采用4800万像素超广角主摄+4800万像素+1300万像素三摄组合，支持10倍光学变焦， 最高60倍数码变焦。前置相机为3200万像素。主摄采用定制索尼 IMX689 传感器。该图像传感器尺寸达 1/1.43 英寸，单像素尺寸 1.12μm ，两者均是所有 48MP 手机摄像头中最大尺寸。四合一后达 2.24μm，也是业内 12MP 中单像素尺寸最大的。相较于上一代产品， IMX 689 感光面积增加 96%，感光灵敏度提升 130%。此外，Find X2 Pro 是行业首款支持 12bit 超清图像采集的手机，支持直出 12bit Raw 数据图片，让专业爱好者拥有更大的后期图片处理空间。同时，Find X2 Pro还是行业首款从拍摄、存储到显示均支持10 bit/10 亿色彩的手机。5G时代视频拍摄越来越受到关注，而Find X2 Pro就带来了更出色的视频拍摄能力。Live HDR功能在大光比与逆光环境下也能够拍摄出无比清晰明亮的画面，彻底解决了逆光场景拍视频时前景过暗或后景过曝的问题，让普通用户也可以拍出电影质感的视频。同时，通过对超广角摄像头拍摄画面进行增稳，Find X2 Pro还带来了超级视频防抖功能，更稳定的画面，更大的取景范围，在后期处理的时候都将带来更大的创作空间。除了单反级别的防抖效果，视频拍摄过程中的对焦速度和准确度无疑也是用户关注的，全像素全向对焦技术将传统单一纵向对焦，升级为横纵向均可对焦的全域对焦，解决了手机拍摄动态画面或暗光场景时，对焦速度慢，且准确率低的问题，大幅提升了普通用户复杂条件下拍照的成片率，不错过每一个精彩瞬间。Oppo Find X2 Pro搭载了骁龙865处理器，双模5G，光感屏幕指纹，IP68级防水防尘，立体声双扬声器。12GB+256GB内存组合，支持UFS3.0闪存，安兔兔跑分607669，Geekbench5单核得分908，多核得分3314。读写速度分别为1752.41MB/s和746.3MB/s。ColorOS 7.1系统，还加入了O1超感画质引擎，包含Motion Clear视频动态插帧和HDR视频画质增强两项技术。此外，该机还支持高通目前最新的X55基带，实现了对持SA/NSA双网、8模37频段的全覆盖。而为了进一步加强Find X2 Pro的5G表现，OPPO为该机使用了360°环绕式天线设计、Smart 5G技术。在日常多种使用场景中，Find X2 Pro均能保持流畅、稳定的5G网速。这一次OPPO为了展示自己的诚意，除了带来极致的性能外，还为Find X2系列专门配备了全新ColorOS系统，带来专业的系统级优化体验。此次，ColorOS 7.1系统延续轻量化设计理念，图标、页面留白和动画交互更加轻巧美观，呈现前所未有的轻盈视觉和轻松交互体验。饱和度更低的色彩模式，持续更新的Art+艺术家合作壁纸，以及全新的全局暗色模式，使得ColorOS 7.1的设计更具美感，交互更直观。5G网络必定导致手机功耗升高，为了保证用户极速的充电体验，快充成为了最不可少的一项。OPPO Find X2 Pro延续了OPPO在快充领域的绝对地位，65W 超级闪充的加持，让OPPO Find X2 Pro只需10分钟就能够从0%充至40%，并且能够充满仅需38分钟。总体来说，这次OPPO Find X2 Pro的升级做到了全方位，从每个参数每个硬件上都有着非常直观的升级，不夸张的说，OPPO Find X2 Pro真正意义上再次刷新了国产旗舰手机的高度。
